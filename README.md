# Software Packaging Automation Bootstrap

This workspace contains a starter multi-repo project structure for a software packaging automation program.

## Included repo skeletons

- `schemas/packaging-standards`
- `frameworks/psadt-enterprise`
- `frameworks/macos-packaging-framework`
- `frameworks/gitlab-ci-templates`
- `deployment/intune-deployment-modules`
- `deployment/terraform-jamf-modules`
- `titles/google-chrome`

Each folder is structured as if it were its own GitLab repository. You can split them into separate repos later or use this workspace as a planning/template source in VS Code.
