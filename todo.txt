Visual Action Builder – Recommended Actions per Lifecycle Phase
Variable Declaration
Custom Variable — PowerShell variable assignment, e.g., $MyVariable = 'Value' (no dedicated PSADT command)

Pre-Installation/Uninstallation
Show Installation Welcome — Show-ADTInstallationWelcome
Show Installation Progress — Show-ADTInstallationProgress
•	Start-ADTProcess with the following parameters exposed in builder: -FilePath -ArgumentList -SuccessExitCodes -RebootExitCodes -PassThru (if specified, expose variable to store result)
•	Start-ADTMsiProcess with the following parameters exposed in builder: -Action -FilePath -ProductCode -ArgumentList -AdditionalArgumentList -Transforms -Patches -SuccessExitCodes -RebootExitCodes -PassThru (if specified, expose variable to store result)
•	Uninstall-ADTApplication with the following parameters exposed in builder: -Name -NameMatch -ProductCode -ApplicationType -FilterScript -ArgumentList -AdditionalArgumentList -SuccessExitCodes -RebootExitCodes -PassThru (if specified, expose variable to store result)
Copy File / Folder — Copy-ADTFile with recommended builder parameters: -Path -Destination -Recurse -Flatten -ContinueFileCopyOnError -FileCopyMode -RobocopyParams -RobocopyAdditionalParams
Copy File to User Profiles — Copy-ADTFileToUserProfiles
Create Folder — New-ADTFolder
•	Delete File(s) (Remove-ADTFile) with following parameters exposed in builder: -Path (accepts wildcards and an array of values) -LiteralPath (does not accept wildcards, will accept an array of values)
•	Delete Folder(s) (Remove-ADTFolder) with -Path and -DisableRecursion parameters exposed in builder.
•	Remove Registry Key / Value (Remove-ADTRegistryKey)
•	Set Registry Key / Value (Set-ADTRegistryKey)
Stop/Kill Process — PowerShell Stop-Process  -Name 
Wait/Delay — PowerShell Start-Sleep (no dedicated PSADT command)
Custom PowerShell — Inline PowerShell script block or direct PowerShell statements (no dedicated PSADT command)
•	Get-ADTPendingReboot “Is System Reboot Pending” by storing Boolean return value from (Get-ADTPendingReboot).IsSystemRebootPending in a specified variable.
